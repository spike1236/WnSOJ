About WnSOJ

This site contains an archive of Olympiad programming tasks with a built-in testing system and a job search platform. Whether you're preparing for competitions or looking for your next career opportunity, WnSOJ has you covered.

Problems: Access our collection of programming challenges
Jobs: Browse job listings and career opportunities

Problems Section

In the "Problems" section, you can solve various programming and math tasks. Each problem has an editorial and solution in C++ language. Platform administrators can add new problems.

How to Submit a Solution
  1. Go to the "Problems" section
  2. Choose a problem that interests you
  3. Paste your code in the code editor
  4. Click the submit button
  5. Wait for the system to run tests and report the verdict
Supported Languages
C++ GNU C++17
g++ source.cpp -std=c++17 -o source
Python Python 3
python3 source.py
Verdicts Explained
Verdict Description
AC Accepted. Your program works correctly and passed all tests. Congratulations!
IQ In Queue. Your solution is in the queue for testing. Please wait.
CE Compilation Error. There's a syntax or compile-time error in your code.
WA Wrong Answer. The output of your program doesn't match the expected output.
RE Runtime Error. Your program terminated with a non-zero return code.
TLE Time Limit Exceeded. Your program runs longer than the allocated time.
MLE Memory Limit Exceeded. Your program uses more than the allocated memory.

Jobs Section

Account Types
Common Account

Business Account

Post, edit, and delete job listings. Wait for qualified specialists to contact you via email or phone number. Perfect for recruiters and companies.
