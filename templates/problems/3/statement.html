Given T integers \(n_1, n_2, n_3, ..., n_T\). For each integer \(n_i\) find the \(n_i\)-th Fibonacci number modulo \(10^{9}+7\).

Suppose \(N\)-th Fibonacci number is \(F_N\). Formula:\(F_{N} = F_{N-1} + F_{N-2}\), \(F_{0} = 1, F_{1} = 1\)

It is guaranteed that queries are valid.

Input:

  • In first line of input is \(T \leq 10^{4}\). In the next \(T\) lines of input there are \(n_1, n_2, n_3, ..., n_T\), \(0 \leq n_i \leq 10^{6}\)

Output:

  • For each query find Fibonacci number.
