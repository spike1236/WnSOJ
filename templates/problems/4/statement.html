Given an undirected graph with \(N\) vertexes and \(M\) edges. Find the vertex that have maximal degree.
If it's more than 1, print the vertex with minimum index.

Input:

  • In first line of input are \(N \leq 10^{5}\) and \(M \leq 10^{5}\). In the next \(M\) lines of input there are \(u\) and \(v\). It's means that there is an edge between \(u\) and \(v\).
    It can be self-loops and multi-edges.

Output:

  • Print answer.
Input:
 
Output:
 
3 4
1 2
1 3
2 3
2 3
 
3
